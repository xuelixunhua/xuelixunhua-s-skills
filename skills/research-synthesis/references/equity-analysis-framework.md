# 权益投资分析框架

## 核心定位

本文档提供权益（股票）投资研究的系统化分析框架，适用于个股研究、行业分析、投资策略构建。

**适用场景**：
- 个股深度研究报告整理
- 行业研究框架构建
- 商业模式与竞争格局分析
- 投资逻辑提炼与验证

---

## 分析方法论框架

### 商业模式评估

商业模式评估是权益投资的核心，决定企业的长期价值创造能力。

#### 蓄积效应分析

**定义**：一件事物前期的发展对后期的结果有必然的正相关影响。

**核心判断标准**：
- ✅ **有蓄积效应**：前期盈利通过复利机制放大未来盈利（如银行、价值投资）
- ❌ **无蓄积效应**：每个项目从零开始，前期盈利对未来无帮助（如房企）

**估值逻辑映射**：
- **PE估值**：适用于有蓄积效应的利润（利润可持续增长）
- **PB估值**：适用于没有蓄积效应的利润（净资产虚高）

**典型案例对比**：

| 维度 | 银行（有蓄积效应） | 房企（无蓄积效应） |
|------|-------------------|-------------------|
| 盈利逻辑 | 挣了钱通过杠杆产生复利 | 挣不挣钱完全看拿地条件 |
| 时间价值 | 前面挣得越多，未来挣得越多 | 前面挣得再多，对未来盈利没帮助 |
| 资产质量 | 真金白银、债权、有价值抵押物 | 净资产很虚，看起来一大堆，说没就没 |
| 商业模式 | 钱生钱，利滚利 | 每一块地都从0开始 |

#### 议价能力分析（核心框架）

**定义**：看利益相关者与上市公司之间谁有求于谁，外在表现是话语权定价权让渡。

**商业模式分级体系**：

| 等级 | 特征 | 代表企业 | 投资价值 |
|------|------|----------|----------|
| **最好-三求** | 上游求+下游求+政府求 | 茅台、苹果、英伟达 | 极高 |
| **次好-两求** | 上游求+政府求（下游不求） | 有色金属、比亚迪 | 高 |
| **再次-一求** | 仅政府求（上下游平等） | 95%以上企业 | 中等 |
| **最差-零求** | 上游不惯、下游不惯、政府不求 | 地产、市区重工业 | 避坑 |

**分析要点**：
- 上游求：原材料供应商挤破脑袋求合作，企业有定价权
- 下游求：消费者排队求购产品，不愁销路
- 政府求：纳税大户，当地政府求别搬走

**投资原则**：
- 同样商业模式下讨论估值才有意义
- 不同商业模式无法横向对比
- 零求商业模式坚决避坑

### 竞争格局分析

#### 核心方法：盯现金流（血条）

**分析逻辑**：
- 竞争本质是消耗战，现金流是企业的"血条"
- 市场份额大的企业为保份额，血条降得快
- 市场份额小的企业同等补贴力度下，血条降得慢
- 时间站在份额低的一方，没有收手必要

**典型案例：电商行业竞争**

| 玩家 | 现金储备 | 市场份额 | 年亏损 | 竞争态势 |
|------|---------|---------|--------|---------|
| 黄马甲 | 1400亿 | 大 | 240亿 | 血条降得快 |
| 蓝马甲 | 1300亿 | 低 | - | 血条降得慢 |
| 红马甲 | 2100亿 | 低 | - | 血条降得慢 |
| 字节 | 4000+亿 | 潜在进入者 | - | 威胁最大 |

**结论推导**：
```
市场份额大 → 想保份额 → 补贴力度大 → 血条降得快
     vs
市场份额低 → 同等补贴 → 血条降得慢 → 时间站在这边
     ↓
只要打下去，份额大的一定输（除非吸小股东血）
```

**破局方法**：
- 收缩战线到30%-40%市场份额
- 与其他玩家保持微妙平衡，互相忌惮

### 产业链分析

#### 寻找优势细分行业

**分析原则**：
- 找世界范围内有竞争优势的细分行业
- 有定价权、垄断性质
- 化工周期里可提高市场份额同时涨价

**中国优势细分行业（化工大类）**：
- 煤化工、磷化工
- MDI、TDI
- 氟化工、硅化工
- 农药、染料
- 电解液、钛白粉

**选股策略**：
- 找边际改善明显（涨价预期高）的龙头企业
- 有股息率更完美
- 缺点：弹性差一点

### 影响联想法（核心方法论）

投资联想法的核心不是简单的相关性，而是**边际效应的放大机制**。

#### 四步推导逻辑

**Step 1: 确定逻辑原点（Trigger）**
- 识别终端市场的趋势性变化
- 例如：房地产回暖、光伏装机高增

**Step 2: 寻找传导路径（Transmission）**
- 确定终端需求向上游原材料的传导路径
- 下游需求占比决定传导强度

**Step 3: 识别刚性瓶颈（Bottleneck）**
- 寻找产能调整周期长、生产连续性强、供给具有刚性的环节
- 例如：玻璃、纯碱、工业硅
- 供给刚性越强，面对需求增长时，价格弹性和爆发力越大

**Step 4: 计算收益弹性（Elasticity）**
- 通过历史数据计算价格传导倍数（Beta值）
- 利用期货的天然杠杆属性，实现收益率的二次放大
- 例如：房地产上升时，玻璃期货弹性达5-8倍，纯碱对玻璃弹性1.2倍

#### 量化模型指标

**必须建立的量化指标**：
1. **产业链渗透率/占比**：下游需求在原材料去向中的占比（如玻璃需求占纯碱下游的65%）
2. **供给刚性系数**：产能扩建周期、停复产成本（冷修周期、固定成本占比）
3. **成本传导敏感度**：原材料在终端产品完全成本中的占比（如纯碱占玻璃成本25%+）
4. **边际参数指标**：
   - **"市种率"（市值/种植面积）**：针对资源型品种，评估单位资源对应的市场估值
   - **期权定价思维**：将合同、优先购买权等非显性权利转化为隐含期权价值

## 基本面分析框架

### 财报分析

**利润表关键指标**：
- **销售收入**：关注行业占比、行业规模、定价权（与市值关系不大，但反映行业地位）
- **毛利率**：制造业超过25%就很好，看成长性和行业地位
- **净利润、净利率**：核心盈利指标

**资产负债表关键指标**：
- 现金资产
- 固定资产
- 无形资产（在建工程、商誉）

**现金流量表**：
- 周转率越高越好
- 经营性现金流/归母净利润：与市净率配合使用的最佳指标

**财务指标分析维度**：
- 截面数据 vs 流量数据
- 单季度值、单季度同比、环比、TTM值、TTM同比

### 估值分析

#### 核心认知

**关键原则**：
- 价值投资 ≠ 低估值风格
- **估值从来都不是决定行情走势的充分条件**
- 估值不会改变股价走势，只会影响后续变化的幅度
- **决定股价拐点的是基本面的景气周期**

**估值的本质**：
```
估值 = 期权属性
    ↓
基本面向上反转 + 估值低位 → 上涨空间大
基本面向下反转 + 估值高位 → 跌幅大
```

**低估值策略有效性**：
- 经济上行期：低估值策略有效（顺周期板块跑赢）
- 经济下行期：低估值策略失效（逆周期板块跑赢）
- 本质：基本面盈利周期而非估值高低决定行情走势

#### 相对估值法

**市盈率（PE）**：
- 公式：PE = 股价/每股收益 = 市值/净利润
- 适用：盈利稳定、周期性弱的行业
- 倒数含义：按现价投资可获得的年度收益率
- 注意：收益为负时无法使用，短期收益无法比较长期增长

**市净率（PB）**：
- 公式：PB = 股价/每股净资产 = 市值/净资产
- 适用：银行业（利润主要来自生息资产）
- 注意：忽略了不同公司资产创造收益的能力差异

**市销率（PS）**：
- 公式：PS = 股价/每股营业收入 = 市值/营业收入
- 适用：高成长行业（TMT、生物医药、网络软件）
- 优点：适用所有公司，不受折旧等影响
- 注意：忽略了成本费用控制能力

#### 绝对估值法

**现金股利贴现模型**：
- 适合分红多且稳定的公司（非周期性行业）
- 公式：P₀ = Σ(DIVₜ/(1+r)ᵗ)
- 挑战：股利无关论（MM定理）

**自由现金流贴现模型**：
- 公式：V = Σ(FCFₜ/(1+WACC)ᵗ)
- 零增长模型：V = FCF/WACC
- 不变增长模型：V = FCF(1+g)/(WACC-g)
- 两阶段/三阶段模型：适用不同成长阶段企业

## 行业分析框架

### 消费股分析框架

消费股分析的核心是理解需求驱动和供给变化。

**关键分析指标（优先级排序）**：

1. **渗透率**（最关键）
   - 行业渗透率 vs 品牌渗透率
   - 四种组合：
     - 最好：行业↑ + 品牌↑（新兴行业龙头，闭眼投）
     - 差一点：行业↑ + 品牌↓（行业上升期，疯狂爆产能）
     - 再差一点：行业↓ + 品牌↑（行业内卷期，如头部白酒）
     - 最差：行业↓ + 品牌↓（如中端白酒）
   - 警惕：渗透率超过50%要警惕，80%、90%没想象空间

2. **人均消费量**（最能找确定性）
   - 对比发达国家，找人均消费量差距最大的地方
   - 个体有差异，群体差异变小
   - 数字上升到亿、十亿量级时更明显

3. **单价**（关注供应端）
   - 不是供应的绝对数量，而是供应在变少还是变多
   - 找产业正在出清、边际改善的行业
   - 只能投卷王（龙一龙二龙三），后排杂毛不考虑

4. **毛利率**
   - 毛利高的行业：成本降低影响不大
   - 毛利低的行业：成本降低影响很大
   - 白酒可以不看（物料成本占比太低），其他行业必须看

**分析公式**：
```
企业净利润 = 营业收入 × 毛利率 - 固定费用
营业收入 = 人数 × 渗透率 × 人均消费量 × 单价
```

### 银行股分析框架

**选股指标（优先级排序）**：

1. **股息率**：高股息未必都能买，但低股息一定排除
2. **地域**：排除房价跌、人口流出省份的地方性银行
3. **经营性现金流/归母净利润**：与市净率配合使用的最佳指标

**投资流派对比**：

| 流派 | 核心逻辑 | 代表标的 | 缺陷 |
|------|---------|---------|------|
| 增速派 | 唯增速论，增速高就买 | 宁波银行 | 经济下行期预测增速困难 |
| 零售派 | 零售业务是未来 | 招商银行 | 关注非息收入 |
| 规模派 | 大就是好/小有潜力 | - | 过于简单 |
| 财报派 | 关注NPL、PCR、ROE | - | 财报真实性存疑 |

**核心认知**：
- 银行商业模式都一样，因此可以比估值
- 净资产来自经营成果，经营成果体现为现金流
- 现金流最好的应有最高市净率

### 医药行业（药店）分析框架

**行业特征**：
- 零售环节，民生重要保障
- 集采压缩利润空间，行业加速内卷
- 关店数量超过开店数量

**投资机会（第一性原理）**：
```
必选消费场景 + 网购冲击小
    ↓
蛋糕变大（人口老龄化）+ 分蛋糕的店变少
    ↓
投资逻辑：数店面
    ↓
店面增加到一定程度 → 对上游有议价权 → 变成2求商业模式
```

**选股原则**：
- 关注加盟店数量增速
- 加盟店本质：把业绩压在员工身上
- 加盟店员工月薪低、加盟费每年都有

## 个股研究分析思路

### 研究流程

**Step 1: 商业模式定性**
- 蓄积效应判断：有/无
- 议价能力评级：三求/两求/一求/零求
- 初步筛选：零求商业模式直接排除

**Step 2: 竞争格局分析**
- 现金流（血条）对比
- 市场份额与竞争态势
- 行业集中度趋势

**Step 3: 产业链定位**
- 上中下游位置
- 成本结构与传导机制
- 是否处于优势细分行业

**Step 4: 基本面验证**
- 财报三表分析
- 估值水平评估
- 与可比公司对比

**Step 5: 投资逻辑提炼**
- 核心驱动因素
- 风险点识别
- 预期收益测算

## 使用说明

### 适用场景判断

**使用权益分析框架的场景**：
- 个股深度研究报告
- 行业研究报告（消费、金融、医药等）
- 商业模式评估
- 竞争格局分析
- 投资逻辑验证

**不适用的场景**：
- 大宗商品基本面分析（使用商品分析框架）
- 宏观经济研究（使用宏观分析框架）
- 纯技术分析、量化策略

### 框架使用原则

1. **有机整合**：根据内容自然逻辑组织，不强制套用所有章节
2. **重点突出**：商业模式、竞争格局、产业链、影响联想是核心
3. **深度优先**：关键概念深度解析（定义→原因→机制→影响）
4. **案例丰富**：保留典型案例和论证逻辑
5. **可追溯**：关键信息注明来源

### 与商品分析框架的区别

| 维度 | 权益分析 | 商品分析 |
|------|---------|---------|
| 核心关注 | 商业模式、竞争格局 | 供需平衡、库存周期 |
| 估值方法 | PE/PB/DCF | 成本曲线、期货定价 |
| 时间维度 | 中长期（3-5年） | 短中期（季度-年度） |
| 关键指标 | 议价能力、现金流 | 库存、开工率、利润 |
| 分析重点 | 企业护城河 | 供需缺口 |
